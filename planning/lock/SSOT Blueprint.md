## 2. Core Sections

### **A. Event Phases (Phase Mechanic)**

| Phase                  | Purpose                           | Stakeholders               | Key Actions                                             | Public View                              |
| ---------------------- | --------------------------------- | -------------------------- | ------------------------------------------------------- | ---------------------------------------- |
| **PRE (Registration)** | Collect registrations, hype event | Admin, Students, Marketing | Registration wizard, team formation, payment collection | Countdown timer, landing page highlights |
| **LIVE (Execution)**   | Manage submissions, judging       | Students, Judges, Admins   | Project submission, live scoring, team collaboration    | Event updates, live dashboards           |

*Notes:* Admin can override phases. All actions during PRE/LIVE/POST are governed by the phase.
